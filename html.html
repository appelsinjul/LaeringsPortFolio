HTML

Hyper Text Markup Languge som også kaldes HTML er det fundament som alle sider på internettet er bygget op på. Det var Tim Berners-Lee, tilbage i 1990 opfandt HTML. Hans mål var at lave et sted hvor alle de forskellige universitets forskere kunne få adgang til hinandens resultater. HTML-projekt blev langt bedre end Tim Berners-Lee overhovedet kunne forstille sig, da det var en af de vigtigste brikker til at skabte internettet.


Element

Et HTML element består af flere dele. Til at starte med skal man altid havde et start og et slut tag, altså i dette tilfælde h1, inde i start taget kan man finde Attributter som kan være forskellige fra tag til tag. Et Attribut tag består af to ting, navnet på den attribut man vælger at bruge og så den værdi attributten har. Efter attributten stopper start taget og det er her man skriver den tekst der skal vise frem på skærmen.

Under min skrivning har jeg brugt to attributter rigtig meget og det er class og id. Forskellen på disse to er at id kan man kun bruge en gang på hver side, da den gør taget unikt, jeg har f. eks. Brugt den til kilder og selve navigationsbaren. Class er lidt anderledes, den kan man bruge på flere steder end bare en gang, jeg har f.eks. valgt at bruge det til mine billeder, da jeg bruger to forskellige størrelser på nogle af billederne.


Semantiske tags

Semantiske tags er tags som både kan læses af maskiner og mennesker, det er tags der fortæller om indholdet, det kan f.eks. være header som er det er øverst på siden, men det kan over være footer, som befinder sine allernederst på siden.

Denne side nedenunder er fra et projekt jeg lavede i samarbejde med Julie omkring andre semantiske tags, bare være opmærksom på der ikke er en tilbage knap til denne side. Semantiske Tags.
